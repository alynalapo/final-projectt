Define a Table
To define a table, one must use the table element, defined by a td and /td tag, holding the content of the table cell. tr and /tr tags represent table rows. To make table header cells, use th and /th tags. theader and tfooter are used in conjunction with one another to group header content and group footer content, respectfully.
Colspan and Rowspan
These specify how many columns or rows a cell should span in a table.
Cell Spacing and Padding
Cellspacing defines the space between cells, whereas cell padding specifies the space betwene the border of a table cell and its contents.
